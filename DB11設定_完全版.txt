DB設定共通　XammpまたはMAMPでサーバー起動XammpならApache起動後、MariaDB(Mysql)起動MAMPはスタートボタンを押すだけWindowsC : \Users\user> cd C:\Users\user\xammp\mysql\binmysql -u root -pPassword : 何も入力しないMACcd /Applications/MAMP/Library/bin./mysql -u root -pPassword : rootSQLファイル配布でxammpp\mysql(\bin)\dataまたはMAMP/Library/binにファイル保存。Mysql>source sqlファイル名;  データベースの生成Mysql>use DataBase_Name;  データベースの有効化（DBの利用）Mysql>select * from Table_Name;（テーブル全体の表示）Mysql>show databases;（DB一覧）Mysql>show tables;（現在使用しているDBのテーブルを表示）Mysql>select database();（現在選択しているデータベースを表示）Mysql>desc Table_Name;　テーブル構造の確認タイプミスで「->」が出た場合、;（セミコロン）を入力することmysql>  exit　終了させるWindows:文字化けが起こる場合VS codeにてエンコードをUTF-8からShift-JISに変更XammpまたはMAMPのサーバー停止、終了
